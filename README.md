# Financial Performance & Profitability Dashboard

## Project Overview
This project analyzes financial performance using Power BI dashboards and business intelligence techniques.

## Features
- Sales & Profit Analysis
- Gross Margin & Operating Margin KPIs
- Expense & Tax Analysis
- Interactive Filters & Slicers
- Multi-page Financial Reporting

## Tools Used
- Power BI
- DAX
- Power Query
- Excel

## Key Insights
- Revenue and net profit demonstrated steady growth.
- Operating margin remained relatively stable over time.
- Expense analysis highlighted major operational cost drivers.
